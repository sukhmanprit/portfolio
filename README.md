# Personal Portfolio Website

This repository contains my personal portfolio website, built to showcase my skills, projects, and professional experience.

## Live Site
https://sukhmanprit.vercel.app/

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Local Development

Run the development server:
```bash
npm run dev
```

Open the site locally at:
```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Customization
- Main page content: `app/page.tsx`
- Global styles and theme tokens: `app/globals.css`
- Hero or profile image: `public/hero.png`

## Deployment

This project is hosted on **Vercel**.  
Any changes pushed to the `main` branch are automatically deployed.

## Notes

This is a personal portfolio project created solely for showcasing my work and experience.  
It is not intended to be a reusable template or an open-source project.
